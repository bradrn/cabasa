count :: (Functor t, Foldable t) => (a -> Bool) -> t a -> Int #

Counts the number of values in a Foldable satisfying a condition. Useful with neighbourhoods e.g. counting the number of live cells in a Moore neighbourhood using count (==Alive) . experiment (moore False)
