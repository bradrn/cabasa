mcell :: Universe Int -> [(String, String)] -> MCell #

A convenient constructor for an MCell value. Since there are only two non-Maybe fields in MCell, this constructor can be used to set all other fields to Nothing, after which other fields can be modified as desired using record update syntax. Usage is something like e.g. (mcell myUn myDiv){ game = Generations }.
