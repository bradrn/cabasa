data CAVals' t #

A record containing all the various bits and pieces of information which the application needs to know to load a new rule.

data CAVals #

If we decide to use a custom state type for our CA, we can't use that type outside of Hint, so this existential wrapper is used to 'hide' the state type from the outside world.
